To run the code, use the following command:
$ python3 12.py <input_file>

Here input_file is euc_100 or noneuc_100 etc.

Or

Alternately the run.sh file can be used, command to use it is:
$ bash run.sh <input_file>

Here input_file is euc_100 or noneuc_100 etc.

Please note that the outout will be printed on terminal itself.
