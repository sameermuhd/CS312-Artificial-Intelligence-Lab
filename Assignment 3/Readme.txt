This assignment as two python codes, clause.py and 12.py.

------------------------------------------------------------------

The clause.py is used to generate the clauses. 
Command to run clause.py:
$ python3 clause.py <number Of Variables> <number Of Clause>

Eg.: python3 clause.py 4 5

Command Line Arguments:
1. clause.py : The file name
2. <number Of Variables> : number of variables to use
3. <number Of Clause> : number of clause to generate

Note:
This file gives clauses.txt as output containing the clause generated.

------------------------------------------------------------------

12.py is the python code with algorithms implementation.
Command to run the 12.py:
$ python3 12.py

Note:
This file takes clause.txt generated by the previous code as input.
It gives output.txt as output containing all relevant information. 

------------------------------------------------------------------

For ease of running please use the given script file run.sh. To run it 
simply use the below command
$ bash run.sh
